## Featured Portfolio Architecture

This repository is engineered as a live portfolio and agency showcase with a clean, vanilla architecture:

- **Static HTML/CSS/JS foundation** with no framework lock-in, optimized for fast rendering and stable responsiveness.
- **Fixed Navbar layout freeze**: the header is intentionally forced to `direction: ltr` so the brand remains on the left and navigation remains on the right across English and Arabic toggles.
- **Native client-side i18n engine**: `script.js` contains the translation data and language toggle logic that switches content seamlessly between English and Arabic without a page reload.
- **Secure contact pipeline**: the contact form is wired to a serverless backend using a dedicated **Cloudflare Worker** to handle submissions safely and avoid client-side mail exposure.

- **`سحب` (Sahab) Power Telemetry Program** — A custom Python utility for real-time electrical stability monitoring and environment profiling using plug-and-play USB hardware identifiers.
